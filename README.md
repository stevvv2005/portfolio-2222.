# BABA Oualid - Engineering Portfolio

Portfolio personnel de BABA Oualid, eleve ingenieur en informatique a l'ENSA Marrakech. Le site presente le parcours, les competences, les projets techniques, les certificats et les informations de contact pour une candidature de stage.

## Objectif

Ce depot sert de vitrine professionnelle. Il met en avant un profil oriente developpement logiciel, cybersecurite, bases de donnees et interfaces web modernes.

## Apercu

- Page portfolio responsive
- Sections profil, competences, projets, formation et contact
- CV telechargeable
- Animations avec Framer Motion
- Interface construite avec Next.js et Tailwind CSS
- Deploiement compatible Vercel

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Installation

```bash
npm install
npm run dev
```

Puis ouvrir :

```text
http://localhost:3000
```

## Scripts

```bash
npm run dev      # Serveur de developpement
npm run build    # Build production
npm run start    # Lance le build Next.js
```

## Structure

```text
app/
  layout.tsx       # Metadonnees et layout racine
  page.tsx         # Page principale du portfolio
components/        # Composants UI et navigation
public/files/      # CV et documents publics
public/solo/       # Images du portfolio
```
